Hardrive

A hard drive, often abbreviated as HDD, is a data storage device found in computers and other
electronic devices. It uses spinning magnetic disks to store and retrieve digital
information. Hard drives offer high-capacity storage for files, applications, and operating
systems, making them a key component in most computers.

Raspberry Pi

A Raspberry Pi is a low-cost, compact computer ideal for DIY projects and learning. It's
versatile, running a Linux-based OS, and features various I/O options. It's popular for
coding, robotics, and home automation.

Learn more

CPU

A CPU (Central Processing Unit) is the core component of a computer, responsible for
executing instructions and performing calculations. It processes data by fetching, decoding,
executing, and storing instructions, enabling software to run and perform tasks.

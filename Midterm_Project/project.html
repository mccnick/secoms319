About the Data

The data visualized in the graph above was obtained using a Raspberry Pi in conjunction with the MPU6050 sensor. This setup allowed us to capture accelerometer data under different testing conditions: when not moving, running, and cycling. We thought this was an interesting way to showcase the power of the Raspberry Pi in addition to our front-end web development skills.
